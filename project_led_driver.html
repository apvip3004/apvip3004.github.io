PWM Led Driver

PWM Led driver with a web application to control time and brightness of lighting.

Used technology:
- C
- HTML/CSS

Equipment:
- ESP32
- led strip
- PWM module
- radiator

Description of project:
The LED lights were constructed from special pink and cold white LEDs. Each LED strip was glued to an aluminium profile, then to an aluminium heat sink. The LEDs were connected to a PWM controller.
