CO2 Controller and PH meter

Controller of CO2 with PH measurment module and touch display.

Used technology:
- Arduino

Equipment:
- Arduino
- Touch display LCD TFT 2.4" with driver ILI9341
- PH measurment module
- relay
- shrimp tank 15l

Description of application:
The application can provide to control of CO2 relay. PH probe is immersed in water. From PH measurment and given KH value can calculate amount of CO2 in fish tank. If CO2 is to small, relay is on. It can be see when animation of bubbles will be active. Application provide set actual value of KH. It can be done by using previously chemical mesuremnt KH and touch the KH description on display. Next, set actual value of KH. Video displays setting the KH value.
