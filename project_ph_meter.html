CO2 Controller and PH meter - 2022

CO2 controller with PH measuring module and touchscreen display.

Used technology:
- Arduino

Equipment:
- Arduino
- Touch display LCD TFT 2.4" with driver ILI9341
- PH measurment module
- relay
- shrimp tank 15l

Description of application:
The application can provide control of the CO2 relay. The PH probe is immersed in the water. The PH measurement and the given KH value can calculate the amount of CO2 in the fish tank. If the CO2 is too low, the relay is switched on. This can be seen when the bubble animation is active. The app allows you to set the current KH value. This can be done by using a previously taken chemical KH measurement and touching the KH description on the display. Setting the current KH value. The video shows the setting of the KH value.
